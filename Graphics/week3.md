# 09.14 (수)
## Vectors
* 1D : [x]
* 2D : [x,y]
* 3D : [x,y,z]
* 4D : [x,y,z,w]
* nD : [x,y,z,w...]

* Typical notation:
  * Column vector
  * Row vector
  
  ![image](https://user-images.githubusercontent.com/68818952/133565354-bbad0810-966c-41a8-8c70-4f072703ecf4.png)
* Meaning:
  * Position (공간 상의 점)
  * Direction (길이를 포함한 방향)
